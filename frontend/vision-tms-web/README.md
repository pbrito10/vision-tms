# Vision TMS Frontend

Frontend React com Vite, em TypeScript.

Validação:

```bash
npm run typecheck
npm run lint
npm run build
```

Views ativas:

- `Run Program`: arranque/paragem do programa e seleção da bancada.
- `Camera Test`: teste da câmara com stream anotado.
- `Bench Config`: criação/edição das bancadas, zonas, sequência e zonas two-hands.
